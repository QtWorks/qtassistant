enum CoolQ::ServiceModule::Result

ServiceModule 执行结果

CoolQ API 密切相关的执行函数都会返回此枚举作为结果表达。

枚举值
NoError 

没有错误（成功）

如果函数执行成功，都会返回当前值。

未知错误

不在当前枚举内的错误值，都会返回 Unknown。
