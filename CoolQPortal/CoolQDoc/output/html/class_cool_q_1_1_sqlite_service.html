◆ Result

enum CoolQ::SqliteService::Result

CqSqlite 执行结果

酷Q API 密切相关的执行函数都会返回此枚举作为结果表达。

枚举值
NoChange 

没有发生改变

如果数据库内容没有发生变化，返回此枚举值。

Done 

已经完成

如果数据库操作执行成功，返回此枚举值。

SqlError 

数据库错误

如果数据库操作执行失败，返回此枚举值。更多信息可以看酷Q的日志。
